AtCoder.Internal.Csr

Description

Immutable Compresed Sparse Row.

Example

>>> import AtCoder.Internal.Csr qualified as C
>>> let csr = build 3 $ VU.fromList @(Int, Int) [(0, 1), (0, 2), (0, 3), (1, 2), (2, 3)]
>>> csr `C.adj` 0
[1,2,3]
>>> csr `C.adj` 1
[2]
>>> csr `C.adj` 2
[3]
