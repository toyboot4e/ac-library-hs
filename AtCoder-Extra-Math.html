power :: Int -> (a -> a -> a) -> a -> a #

\(O(\log n)\) Calculates \(s^n\) with custom multiplication operator using the binary lifting technique.

The internal implementation is taken from stimes, but powers uses strict evaluation and is often much faster.

Constraints

  • \(n \gt 0\)
